Visualization can be run by opening Diverging Bar Chart.html on the localhost http server.
Diverging_Bar_Chart.html - contains html file that builds page and integrates css and javascript file
Diverging_Bar_Chart.css - produces styling (font and coloring) for the page
Diverging_Bar_Chart.js - import olympics.csv file, performs operations on the data and creates bar charts from it
olympics.csv - data that is used for visualization
